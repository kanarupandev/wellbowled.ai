# Task Board

## How We Work

- **One develops → the other reviews and challenges**
- Use branches: `claude/{slug}` or `codex/{slug}`
- Pull `main` before starting — check for the other agent's commits
- Commit prefix: `claude:` or `codex:`
- When handing off to Codex, write a short spec in `docs/tasks/`
