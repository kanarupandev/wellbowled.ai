# Bowling Signature DNA Feature — Site Map

## Purpose

This folder defines the Bowling Signature DNA feature as a product/design concept.

The feature is intended to:
- identify a user's nearest bowling archetype
- classify a bowling action using phased action descriptors
- support retrieval against a pilot pool of prominent bowlers
- later scale toward a large archetype/cluster library

This is a design and research folder, not an implementation folder.

## Reading Order

### 1. `WHY.md`
Explains why this feature matters for the business, product, and coaching value.

### 2. `WHAT.md`
Defines what the feature actually is, what it stores, and what the pilot should contain.

### 3. `HOW.md`
Defines how the pilot should work: phase model, parameter model, retrieval design, chunking, and source strategy.

## Core Product Statement

The Bowling Signature DNA feature is not just a similarity gimmick.
It is intended to become:
- an identity product
- a coaching reference engine
- an archetype retrieval system
- a long-term data moat for bowling analysis

## Current Scope

Current intended scope:
- seam bowling first
- top 10 prominent bowlers as pilot
- around 100 action-signature parameters defined at taxonomy level
- retrieval based primarily on action-phase signature, not career achievement
- future expansion toward around 1000 archetype clusters

## Key Rule

**Nearest-bowler retrieval should be driven mainly by action signature, not by biography or fame.**
