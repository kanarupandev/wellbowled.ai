## Changes from v0.0.1

1. **Hook changed:** "This doesn't look right." → "Why does this work?" — steers toward curiosity, avoids legality/chucking debate.
2. **Overclaiming removed:** "Both devastating at 145+ km/h" → "Both elite. Both effective." — defensible without speed claims from nets footage.
3. **Phase matching treated as unverified:** Frame IDs are now explicit placeholders. No "same moment" claim. Side-by-side described as "closest equivalent delivery phase."
4. **Back half simplified:** Two side-by-sides → one hero side-by-side. Beat 4 now lands the takeaway cleanly instead of introducing a second comparison.
5. **Production notes separated:** All clip quality concerns, open decisions, and risks moved to `production_notes_v0.0.2.md`.
