# Content Production Plan

## Goal

Create wow-factor bowling content that is:

- useful
- visually premium
- repeatable
- mostly automatable
- achievable with not more than 1 hour of manual intervention per asset

## Core Format Stack

Start with 4 repeatable formats only.

### 1. Release Frame

Best for:

- short hooks
- premium feel
- low manual effort

Output:

- one freeze frame
- 2-3 labels
- 15-30 seconds

### 2. Why This Works

Best for:

- one bowler
- one visible mechanic
- one takeaway

Output:

- 20-45 seconds
- 3 key moments max

### 3. Frame Battle

Best for:

- side-by-side wow factor
- comparing two bowlers at the same phase

Output:

- 20-45 seconds
- same phase
- same scale
- same visual language

### 4. Energy Flow

Best for:

- signature format
- highest brand differentiation

Output:

- 30-90 seconds
- 3-4 anchors max
- use sparingly as premium content

## Manual Intervention Budget

Target max manual time per asset:

- clip selection: 10-15 minutes
- best frame/phase selection: 10-15 minutes
- overlay cleanup and label tuning: 15-20 minutes
- caption/title/final check: 10 minutes

Total target:

- 45-60 minutes

## Recommended Workflow

1. Pick format first, not bowler first.
2. Pull one clip with a clear side-on or near-side-on view.
3. Extract key frames automatically.
4. Select the best 1-3 frames manually.
5. Apply standard overlay template.
6. Keep on-screen text minimal.
7. Export vertical first.
8. Reuse the same structure across the series.

## What To Avoid

- too many metrics on screen
- custom visual design for every asset
- more than 3 insights per video
- overclaiming biomechanics certainty
- long manual polishing loops

## Recommended Publishing Sequence

Phase 1:

- 3 Release Frame videos
- 2 Why This Works videos

Phase 2:

- 2 Frame Battle videos
- 1 Energy Flow video

Phase 3:

- review what performed
- double down on the best 2 formats

## Success Criteria

- strong first-frame visual
- one clear idea per video
- recognizable repeated format
- low manual overhead
- content can be produced consistently, not just occasionally
