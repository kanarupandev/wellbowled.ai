# Product Roadmap

## Product Thesis

The company should not be framed as a single analysis tool.
It should be framed as a bowling-specialized product ecosystem.

## Core Service Stack

### 1. DNA Signature
Purpose:
- identity product
- acquisition layer
- shareable top-of-funnel offer

Example outputs:
- who do you bowl like
- nearest elite archetypes
- bowling signature profile
- phase-by-phase similarity and difference summary

### 2. Extracted-Figure Analysis Techniques
Purpose:
- visual differentiation layer
- content engine
- modular analysis outputs

Example outputs:
- phase overlays
- release-shape analysis
- sequencing visuals
- comparative silhouette or pose outputs
- visual technique reports

### 3. Detailed Coach Report
Purpose:
- premium monetization layer
- coach-facing serious utility
- deeper interpretive product

Example outputs:
- coaching analysis report
- elite comparator references
- technical breakdowns
- strengths, deviations, development areas

## Service Logic

Recommended structure:
- DNA Signature = acquisition
- extracted-figure analysis = differentiation
- coach report = monetization core

## Roadmap Direction

### Phase 1
- seam bowling only
- strongest DNA/signature MVP
- limited but high-quality extracted-figure techniques
- premium coach report offer

### Phase 2
- richer signature pool
- better comparisons
- stronger visual product suite
- possible progress tracking and player history

### Phase 3
- workflow products
- clip management
- coach tools
- broader structured bowling intelligence products

## Seam First, Spin Later

Seam bowling should be the starting scope.
That is the right sequencing.

Reason:
- clearer mechanics
- stronger early content hook
- easier DNA/signature product framing
- simpler first ontology

Spin should be treated as:
- future expansion
- separate ontology family
- separate comparison logic

It should not be forced into the first version if quality is not yet strong.

## Key Product Rule

**Start narrower in product scope than in brand scope.**
