# wellBowled.ai Content Production

Social media bowling analysis clips — planning, scripts, assets, and exports.

## Directory Structure

```
content/
  plan.md              — strategy, tools, workflow
  scripts/             — per-clip scripts (hook, breakdown, payoff)
  raw/                 — raw footage (gitignored)
  exports/             — final rendered clips (gitignored)
  assets/              — overlays, fonts, templates, thumbnails
```

## Quick Start

1. Read `plan.md` for tool setup and workflow
2. Create a script in `scripts/` for each clip
3. Drop raw footage in `raw/`
4. Export finals to `exports/`
