# 3-Minute Demo Blueprint (Judge-Optimized)

## 0:00-0:15 - Hook
- One-sentence problem.
- One-sentence user.
- One-sentence value.

## 0:15-0:50 - Live Magic Moment
- Show user starting live session.
- Show Gemini Live audio feedback in real-time.
- Show count/feedback updates in UI.

## 0:50-1:35 - Deep Clip Intelligence
- Show automatic 5s clip capture after delivery.
- Show deep analysis card (pros/cons/injury risk + pace band).
- Show color-coded timeline markers tied to model labels.

## 1:35-2:10 - Explainability & Control
- Ask phase-focused chat question.
- Jump to exact phase timestamp in clip.
- Show evidence-linked answer (not vague coaching text).

## 2:10-2:35 - Technical Proof
- 1 architecture slide: Live API + on-device clip + deep analysis pipeline.
- Mention fallback behavior for latency/network/model issues.

## 2:35-3:00 - Impact + Close
- Who benefits and why now.
- Measurable expected outcome.
- Crisp close: what is shipped today and what comes next.

## Delivery rules
1. Show product, not talking heads.
2. Keep overlays legible.
3. No fake speed edits that misrepresent runtime behavior.
4. Make Gemini role explicit in narration.
